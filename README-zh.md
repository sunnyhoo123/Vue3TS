# vue3-plus

Vue3 + TS + Webpack5

- Ant Design of Vue(组件)
- Pinia(状态管理库)

此项目没有用Vite，而是webpack

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```

### Lints and fixes files

```
pnpm run lint
```
