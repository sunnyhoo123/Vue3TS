# vue3-ts-webp

Vue3 + TS

- Ant Design of Vue(组件)
- Pinia(状态管理库)
- Webpack

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```

### Lints and fixes files

```
pnpm run lint
```
